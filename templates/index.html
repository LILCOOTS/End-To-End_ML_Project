About This Project

This machine learning project focuses on predicting student performance across three key academic areas: mathematics, reading, and writing. By analyzing various demographic and educational factors, our model provides insights into what influences student success.

The model considers factors such as gender, race/ethnicity, parental level of education, lunch type, and test preparation course completion to generate accurate predictions. This information can be valuable for educators, parents, and educational institutions to understand and improve student outcomes.

Our end-to-end machine learning pipeline includes data ingestion, preprocessing, feature engineering, model training, and deployment through a user-friendly web interface.
